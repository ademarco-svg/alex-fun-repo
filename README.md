# Lift Log

A mobile-first React lifting tracker for a 12-week, 4-day training block. Workout
selection and per-set weights are saved automatically to `localStorage`, so the
app works well as a quick iPhone browser companion in the gym.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Use it on your phone

This app is set up to publish to GitHub Pages after the pull request is merged
into `main`.

Once GitHub Pages is enabled for the repository, open:

```text
https://ademarco-svg.github.io/alex-fun-repo/
```

On iPhone, tap Share, then Add to Home Screen to make it feel like a normal app.
